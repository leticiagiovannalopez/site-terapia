# Terapia — Website Institucional

Website institucional **responsivo e mobile-first**, desenvolvido para apresentação de serviços de **terapia cristã**, com foco em clareza, conversão e experiência do usuário.

O projeto prioriza **design limpo**, **hierarquia visual**, **animações sutis** e **boas práticas de desenvolvimento frontend**, mantendo acessibilidade, identidade visual consistente e performance.

---

## Sobre o Projeto

Este site foi desenvolvido como um **produto real**, com objetivos **comerciais e institucionais**, simulando um cenário profissional completo — desde o design visual até a implementação técnica.

O website apresenta informações sobre a terapeuta, serviços oferecidos, planos, depoimentos e chamada para agendamento, com navegação fluida, layout moderno e adaptação total para dispositivos móveis.

---

## Tecnologias Utilizadas

* **HTML5**
  Estrutura semântica, acessível e bem organizada.

* **CSS3**
  Flexbox, Grid Layout, animações, transições e responsividade avançada.

* **JavaScript (Vanilla)**
  Manipulação do DOM, eventos de scroll, controle de estados visuais e animações.

* **Intersection Observer API**
  Utilizada para animações suaves conforme os elementos entram na viewport, com melhor performance.

* **SVG**
  Elementos vetoriais decorativos leves e escaláveis.

## Aprendizados

Durante o desenvolvimento deste projeto, foram aprofundadas habilidades em:

* Design responsivo e mobile-first
* Animações com CSS e JavaScript
* Manipulação avançada do DOM
* Intersection Observer API
* Organização e estruturação de código
* UX/UI focado em conversão e clareza de informação
